# Short Workshop Scratch Course (4-8 Weeks)

**Target Audience:** School children ages 8-12
**Duration:** 4-8 weeks (8-16 hours)
**Frequency:** 1-2 hours per week (or intensive: 2+ hours per day)
**Level:** Beginner Introduction

---

## Course Overview

This condensed 4-8 week workshop introduces students to the core concepts of Scratch programming. Perfect for after-school programs, summer camps, or short-term learning experiences. Students will create their first interactive projects and gain confidence with programming fundamentals.

---

## Learning Objectives

By the end of this workshop, students will be able to:
- Understand basic programming concepts
- Navigate the Scratch interface
- Create simple interactive projects
- Use motion, events, and conditionals
- Understand the importance of loops and variables
- Share their creative work

---

## Workshop Structure

### Format Options

**Option A: Weekly Meetings (4-8 weeks)**
- 1-2 hours per week
- 4-8 weeks total
- Pace: Relaxed, with review time

**Option B: Intensive Workshop (1-2 weeks)**
- 2+ hours per day
- 5 consecutive days or 2 weekends
- Pace: Fast, focused learning

---

## Session 1: Getting Started with Scratch (1-2 hours)

**Objectives:**
- Set up Scratch account
- Explore the interface
- Create first working program

**Content:**
- Welcome and icebreaker
- Scratch account setup (5 min)
- Interface tour (10 min)
- Blocks and categories (5 min)
- First script: Make sprite move (15 min)
- Extension: Keyboard controls (optional)

**Hands-On Activity:**
- Create program to move sprite with mouse or keys
- Add sound effect

**Project:** "Moving Sprite"
- Sprite moves when script runs
- Includes visual or sound feedback

**Wrap-up:**
- Share and celebrate first projects
- Q&A

---

## Session 2: Events and Interaction (1-2 hours)

**Objectives:**
- Understand event-driven programming
- Create interactive responses
- Combine multiple blocks

**Content:**
- Review of Session 1 (5 min)
- Event blocks deep dive (10 min)
- Creating multiple scripts (10 min)
- Hands-on: Build interactive character (30 min)

**Hands-On Activity:**
- Create sprite that responds to different events
- Add sound effects and visual changes
- Multiple keyboard responses

**Project:** "Interactive Character"
- Responds to at least 3 different user actions
- Changes costume or appearance
- Includes sound effects

**Wrap-up:**
- Demo projects to class
- Discuss what made interactions work

---

## Session 3: Motion, Loops, and Animation (1-2 hours)

**Objectives:**
- Understand loops and repetition
- Create animations
- Use loops efficiently

**Content:**
- Review: Chains of blocks (5 min)
- Introduction to loops (10 min)
- Repeat vs. Forever blocks (10 min)
- Creating animations (20 min)

**Hands-On Activity:**
- Create simple animation (walking sprite)
- Create rotating sprite
- Build automated movement pattern

**Project:** "Animated Sprite"
- Smooth animation using loops
- At least 2 different animation sequences
- Uses repeat and/or forever blocks

**Wrap-up:**
- Share animations
- Discuss efficiency of loops

---

## Session 4: Multiple Sprites and Scenes (1-2 hours)

**Objectives:**
- Work with multiple sprites
- Create simple scenes
- Understand sprite coordination

**Content:**
- Adding multiple sprites (5 min)
- Managing individual sprites (10 min)
- Creating scenes (10 min)
- Building interactions (15 min)

**Hands-On Activity:**
- Create scene with 3+ sprites
- Each sprite has its own behavior
- Simple interactions (sprites respond to clicks)

**Project:** "Interactive Scene"
- 4+ different sprites
- Each sprite responds to interactions
- Organized and visually appealing
- Optional: Multiple scenes

**Wrap-up:**
- Gallery walk of projects
- Peer feedback

---

## Session 5: Simple Game Logic (1-2 hours)

**Objectives:**
- Use basic conditionals
- Introduce scoring
- Create simple game mechanics

**Content:**
- If/else blocks (10 min)
- Simple conditionals (10 min)
- Scoring introduction (10 min)
- Building simple game (20 min)

**Hands-On Activity:**
- Create simple click/catch game
- Add scoring system
- Implement simple win condition

**Project:** "Simple Game"
- Click or catch mechanic
- Score tracking
- Simple end condition
- 2+ sprites (player and target/obstacle)

**Wrap-up:**
- Playtest each other's games
- Discuss game mechanics

---

## Session 6: Variables and Game Features (1-2 hours)

**Objectives:**
- Understand variables
- Implement lives/health system
- Add game polish

**Content:**
- What are variables? (10 min)
- Creating and using variables (10 min)
- Variables in games (10 min)
- Game polish techniques (15 min)

**Hands-On Activity:**
- Add lives system to previous game
- Implement game over condition
- Add visual feedback and polish

**Project:** "Improved Game"
- Score and lives tracking
- Proper game over condition
- Visual polish (colors, effects, sounds)
- Playable and fun

**Wrap-up:**
- Share improved games
- Discuss what made games better

---

## Session 7: Design Your Own Project (1-2 hours)

**Objectives:**
- Plan own project
- Apply learned concepts
- Practice problem-solving

**Content:**
- Project planning (10 min)
- Review of available blocks (10 min)
- Building and debugging (30-40 min)

**Hands-On Activity:**
- Choose project type (game, story, animation, interactive art)
- Plan project concept
- Build using learned techniques
- Troubleshoot issues

**Project Options:**
- **Game:** Create original game concept
- **Story:** Interactive narrative with multiple scenes
- **Animation:** Animated sequence or scene
- **Art:** Interactive visual creation

**Wrap-up:**
- Begin presentations
- Q&A for creators

---

## Session 8: Presentations and Celebration (1-2 hours)

**Optional - can combine with Session 7 if time is limited**

**Objectives:**
- Present and share work
- Celebrate learning
- Reflect on growth

**Content:**
- Project presentations (2-3 min each)
- Peer feedback and comments
- Reflection and Q&A
- Certificate presentation (optional)

**Activities:**
- Each student demos their project
- Explain what they created and how
- Answer questions from peers
- Give feedback to others

**Wrap-up:**
- Discuss favorite concepts learned
- Next steps for continued learning
- Celebrate everyone's accomplishments

---

## Compressed Workshop Schedule

If running as intensive 1-2 week workshop:

### 5-Day Intensive (2-3 hours per day)

**Day 1:** Sessions 1-2 (Getting Started & Interaction)
**Day 2:** Sessions 3-4 (Loops & Multiple Sprites)
**Day 3:** Session 5 (Simple Games)
**Day 4:** Session 6 (Variables & Polish) + Session 7 (Own Project Planning)
**Day 5:** Session 7-8 (Project Building & Presentations)

---

## Assessment Methods

1. **Daily Projects** - 60%
   - Each session produces working project
   - Shows application of concepts
   - Creative expression

2. **Participation** - 20%
   - Active engagement
   - Attempting challenges
   - Helping peers

3. **Final Project** - 20%
   - Complexity and creativity
   - Presentation quality
   - Ability to explain work

## Tips for Workshop Instructors

✅ **Pacing:** Keep energy high and move through topics
✅ **Flexibility:** Adapt pace based on student needs
✅ **Support:** Pair struggling students with buddies
✅ **Inspiration:** Show example projects to spark ideas
✅ **Celebrate:** Praise effort and creativity throughout
✅ **Safe Environment:** Make it okay to make mistakes
✅ **Share:** Regular project sharing maintains enthusiasm
✅ **Resources:** Have backup projects for quick finishers
